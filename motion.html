Visually Communicating Through Motion

All of my design approaches have a storytelling element to them. Envisioning a story through market research and empathy helps a brand connect to the target audience.

This project was all done in-house. All of the assets were created by myself to avoid any copyright infringement. A short video that lets clients know the assets I bring to any design team.

This particular project added a tactile element to create an animation that had a brand message attached. I chose to do an endangered wildlife story involving WWF.

A project using After Effects where one word was chosen and the task was to describe the word visually. I chose the word syzygy, which has roots in astronomy, and created all the assets for the video myself aside from the light saber plugin and sound effects.

A UI animation for a prototype app that was created last term. Using After Effects the task was to create a UI animation following the Material Design standards from Google.
